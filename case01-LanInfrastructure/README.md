# Case 01 — LAN Infrastructure 

Vertex Solutions requires an initial enterprise campus infrastructure capable
of supporting approximately 100 users and future network services.

## Objectives

- Build the initial three-tier campus topology
- Create the management VLAN
- Assign unique management addresses
- Configure MOTD banners
- Document cabling and endpoint placement
